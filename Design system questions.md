# Design System Questions

## 1. Font Sizes for Each Type of Text

- **Main Title (Page Title):**
  - Font Size: 2rem (32px)
  - Font Weight: 700 (Bold)
  - Example Usage: Main page headings

- **Card Title:**
  - Font Size: 1.25rem (20px)
  - Font Weight: 600 (Semi-bold)
  - Example Usage: Card headers, section titles

- **Body Text:**
  - Font Size: 1rem (16px)
  - Font Weight: 400 (Regular)
  - Example Usage: Main content, descriptions

- **Small Text / Caption:**
  - Font Size: 0.875rem (14px)
  - Font Weight: 400 (Regular)
  - Example Usage: Labels, helper text, captions

## 2. Font Styles Details

- **Font Family:**
  - Primary: Inter, Arial, sans-serif
- **Font Weights:**
  - 400 (Regular)
  - 500 (Medium)
  - 600 (Semi-bold)
  - 700 (Bold)
- **Line Height:**
  - Titles: 120%
  - Body: 150%
- **Letter Spacing:**
  - Normal (default)

## 4. Width and Height of Cards and Other Components

- **Card Component:**
  - Width: 320px (default, can be responsive)
  - Height: 180px–auto (depends on content)
  - Padding: 24px
  - Border Radius: 12px

- **Button:**
  - Height: 40px
  - Padding: 0 16px
  - Border Radius: 8px

- **Input Field:**
  - Height: 40px
  - Padding: 0 12px
  - Border Radius: 8px

- **Modal/Dialog:**
  - Width: 480px (default)
  - Padding: 32px
  - Border Radius: 16px

> **Note:** If you need specific dimensions for any other component, please specify the component name and context.
